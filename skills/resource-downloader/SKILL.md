# ⚡ resource-downloader 智能下载技能说明书

高性能、防封锁、多模式并存的系统级资源安全高速下载技能。通过本地脱敏管道与自愈引擎实现全流程无感资源交付。

## 📖 技能介绍
本技能利用本地静态分流引擎和底层并发垂直爬虫，自动过滤、脱敏、打分并唤起本地默认下载软件。
根据传入的资源类型（影视、软件等），本技能将以不同策略启动底层的下载工具，极速解决多媒体影视获取及安全软件官方包安装问题。

## 🛠️ 输入参数
* `title` (str, 必须): 待下载的资源名称（如 `"奥本海默"`、`"Google Chrome"`）。
* `type` (str, 可选): 资源类型分类。支持：
  - `"movie"`: 激活并发种子群搜评分，并唤起迅雷高速下载。
  - `"software"`: 激活传统网页搜索引擎，寻获官网下载地址并通过直链安全下载。
* `quality` (str, 可选): 期望影视画质等级，默认为 `"1080p"`。

## 🚀 执行流程
1. **静态洗白**：意图匹配并格式化后，将核心资源名称与类型参数投递给该技能。
2. **多模式下载**：
   - **影视类 (`type="movie"`)**：并发请求 3+ 种子站，提取哈希并进行统一质量/擦边过滤，择优唤起迅雷；若无则启动浏览器及 Web 搜索（不包含 AI 联网搜索）双重兜底。
   - **软件类 (`type="software"`)**：通过常规静态搜索引擎安全寻获官方下载链接，启动多线程直链高速下载。
3. **视觉自愈控制**：在 Windows 平台上，激活屏幕像素分析定位“立即下载”蓝色按钮并模拟点击实现自愈闭环。
